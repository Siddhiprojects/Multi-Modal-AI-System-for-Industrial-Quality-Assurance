# Multi-Modal-AI-System-for-Industrial-Quality-Assurance
Automate a manual industry process.
<br>
<h4>Project Description</h4>
<br>
In heavy manufacturing, missing a tiny scratch on a steel sheet can lead to catastrophic structural failures. Currently, human inspectors stare at metal all day and manually write damage reports. We are going to automate this entirely by building a dual-model "Vision-to-Language" AI pipeline that acts as an autonomous Quality Assurance (QA) officer. This system gives our AI both "eyes" and a "brain." First, we will train a YOLOv8 computer vision model on real factory data to spot microscopic flaws like cracks and scratches on hot-rolled steel. Instead of just drawing a bounding box, this model will pass its findings to an LLM. The LLM will process the visual data and automatically draft a professional, formatted maintenance ticket explaining the defect and the recommended repair, all wrapped up in a clean Streamlit web app. What You Will Learn Python, NumPy, Pandas, Deep Learning & Model Training, Computer Vision, LLM Integration, System Designing
